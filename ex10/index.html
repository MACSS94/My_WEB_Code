Block versus Inline Elements

Generally, block-level elements may contain inline elements and other block-level elements. Generally, inline elements may contain only data and other inline elements. Inherent in this structural distinction is the idea that a block elements create "larger" structures than inline elements.

Block Elements

São elementos que definem blocos autónomos de um documento, sendo peças chave na definição
da sua estrutura.

Os exemplos mais vulgares de Block Elements são os relacionados com os títulos, os parágrafos,
as listas e o contentor genérico div.

Inline Elements

São os elementos de um nível estrutural inferior, que habitam no interior de bloc-elements. Os exemplos mais vulgares de Inline Elements são os elementos de uma frase (phrase elements), hiperlinks, imagens e os controlos dos formulários.
